# DiaryAPP by Welile Dlamini

A personal multimedia diary application for capturing life's moments.

## Features
- 📝 Write text entries
- 📸 Add photos from camera/gallery
- 🎙️ Record voice notes
- 🔐 PIN security protection
- 🌤️ Automatic weather information
- 🔔 Customizable reminders
- 📊 Statistics and streaks
- 🌓 Light/Dark theme

## Technologies Used
- Flutter 3.x
- Isar Database (local NoSQL)
- OpenWeatherMap API
